O que é o HTML?

HTML é uma linguagem de marcação usada para estruturar paginas na web. Ele define titulos, paragrafos, imagens, links, etc.

O que é o CSS?

Css é uma linguagem usada para estilizar paginas HTML.Ele controla cores, fontes, espaçamentos e o layout dos elementos.
